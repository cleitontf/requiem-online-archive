About this Project

Origin

requiem.isnet.ru was a community-maintained knowledge base for Requiem Online, a free-to-play horror MMORPG. When the site went offline following the Russian server shutdown, its content was retrieved from the Wayback Machine and rebuilt as a fully static archive.

No server-side code, no database, no login system — everything runs directly in the browser. Read more about the game at About the Game →

What's preserved
FeatureDetails
Skill CalculatorAll 24 playable classes, with skill trees, DNA allocation, and shareable build links via URL.
Monster Database866 monsters across 37 zones, with maps and clickable markers.
Quest Guide1,768 quests with filters, objectives, NPC locations, and rewards.
CommunityActive links to Discord, YouTube, and fan sites.
What was removed
SectionReason
/experience/Depended on XP tables linked to a database that wasn't preserved.
/fitting/Depended on equipment data and server endpoints that are no longer available.
/weaponry/Depended on a weapons database not included in the Wayback Machine archive.
Known limitations
  • Content was recovered from the Russian site requiem.isnet.ru — some translations may be imprecise.
  • Individual monster images aren't available — a generic icon is shown instead.
  • Quest and seasonal event dates reflect the state of the Russian server at the time of archiving.
  • Raw data files (Quest.csv, ItemAsset.csv) aren't included in the repository.

How to run
In the browser
Hosted on GitHub Pages — no installation needed.

Open online →
Locally (Windows / Linux)
1. Install Python 3 — on Windows, check "Add Python to PATH" during installation

2. Extract the project zip to any folder

3. Open a terminal in that folder and run:
python serve.py

The browser opens automatically at localhost:8000
